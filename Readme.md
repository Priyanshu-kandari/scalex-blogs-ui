# ScaleX — Insights & Blogs UI

A sleek, dark-themed **Blogs & Insights** page inspired by modern SaaS websites.  
Built with pure HTML & CSS.

## Features

- Dark UI with orange accent colors
- Category filter pills (All, Tech, SaaS, Marketing)
- Responsive blog card grid with image thumbnails
- Clean navbar with CTA button

## Tech Stack

- HTML
- CSS

## Getting Started

Clone the repo and open `index.html` in your browser.

```bash
git clone https://github.com/yourusername/scalex-blogs-ui.git
cd scalex-blogs-ui
open index.html
```
